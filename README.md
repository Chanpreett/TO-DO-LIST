# TO-DO LIST APP 📝

A simple To Do List app built with HTML, CSS and JavaScript.

## Features
- ✅ Add tasks
- ✅ Delete tasks
- ✅ Strike through completed tasks
- ✅ Enter key support
- ✅ Local Storage (tasks saved after refresh)

## Upcoming Features

- 🔄 Due dates
- 🔄 Progress bar
